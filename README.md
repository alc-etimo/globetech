# Portfolio: Localisation QA Engineer (Cork, Ireland)

## 📌 About this portfolio

Online portfolio developed for the **Localisation QA Engineer with Portuguese** role at GlobeTech in Cork, Ireland (onsite position). The portfolio demonstrates 10+ years of experience in software, hardware, and device localisation testing.

## 🎯 Target role

- **Position:** Localisation QA Engineer
- **Location:** Cork, Ireland (onsite 5 days/week)
- **Focus:** Software testing, UI/UX validation, hardware testing, automation readiness

## 🧩 Sections

| Section | Content |
|---------|---------|
| **Header** | Fixed navbar with logo and anchor links |
| **Hero** | Slideshow with impact phrases + grey overlay |
| **Service Wrapper** | 3 columns: About Me, My Approach, What I Offer |
| **Who I Am** | Contextual text about experience and skills |
| **Case Studies** | 4 cases: Bug Reporting, Cross-Platform, Beta Testing, Subtitle QA |
| **Skills** | Technical competencies grid |
| **Relocation** | Cork relocation readiness information |
| **Collaborations** | Slideshow with client logos and roles |
| **Social & Footer** | Contact links and credits |

## 🛠️ Case studies included

1. **Systematic Bug Reporting on macOS & iOS** — structured reporting with reproduction steps and screenshots
2. **Cross-Platform Localisation Testing** — test coverage matrix for 4+ platforms
3. **Functional Beta Testing** — beta environment testing for software suite
4. **Subtitle QA & Error Taxonomy** — error taxonomy design for LLM training data
